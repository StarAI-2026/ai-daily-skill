# 封面风格：杂志编辑风

> **风格定位**：平面设计感、色块拼接、大字报排版。适合媒体号、资讯类。
> **视觉重心**：大字标题 + 色块分割，无人物。

## 基础固定规则

生成一张横版 16:9 封面，杂志编辑设计风格，色块拼接，大字排版，平面设计感强。

视觉优先级：主标题 > 色块分割 > 副标题 > 编号/日期。

## 标题生成规则

### 字体风格
主标题使用**超粗压缩无衬线字体**：
- 字形极粗、紧凑、有压迫感
- 大面积占据画面
- 颜色与背景色块形成强对比

副标题使用**细体无衬线**，搭配编号或日期。

### 色彩方案
每次随机选1：
- 红底白字 + 黑色色块
- 黄底黑字 + 蓝色色块
- 蓝底白字 + 橙色色块
- 黑底黄字 + 白色色块

## 背景样式
- 纯色底 + 几何色块拼接
- 色块可为矩形/三角形/对角分割
- 可选：报刊纹理、半色调点

## 装饰元素（随机选1-2）
- 编号数字（大号、半透明）
- 日期标注
- 极简线条分隔
- 标签贴纸效果

## 右上角气泡
纯色色块标签，文案根据日报类型。

## 最终校验规则
- [ ] 大字标题占据主要画面
- [ ] 色块拼接清晰
- [ ] 平面设计感强
- [ ] 无人物
- [ ] 排版有节奏感
