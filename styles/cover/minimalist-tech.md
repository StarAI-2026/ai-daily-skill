# 封面风格：极简科技

> **风格定位**：干净、克制、留白为主。适合科技博主、专业媒体调性。
> **视觉重心**：标题居中，几何线条辅助，无人物。

## 基础固定规则

生成一张横版 16:9 封面，整体为极简科技风格，大量留白，几何线条辅助，高对比度但克制。

视觉优先级：主标题 > 几何装饰 > 副标题 > 底部署名条。

画面中不出现任何人物。标题是唯一视觉重心。

## 标题生成规则

### 标准结构
- 主标题：直击核心事件，短平快
- 副标题：补充细节或边界

### 字体风格
主标题使用**粗体无衬线字体**：
- 字形厚重、干净、现代
- 大号字体居中或偏左
- 颜色为纯白或品牌主色

副标题使用**细体无衬线**，灰色或半透明白色。

### 色彩方案
每次随机选1：
- 深蓝底 + 白字 + 蓝色线条
- 纯黑底 + 白字 + 橙色点缀
- 深灰底 + 白字 + 青色线条
- 墨绿底 + 白字 + 金色点缀

## 背景样式
- 纯色深底，微弱渐变
- 可选：细网格线、电路纹理、数据流线条（低透明度）
- 四周轻微暗角

## 装饰元素（随机选1-2）
- 几何线条（直线/圆弧/折线）
- 半透明圆环
- 极简图标（与主题相关）
- 数据点连线图

## 右上角气泡
画面右上角浅色半透明气泡，文案根据日报类型：
- AI 日报：`StarAI日报`
- GitHub 日报：`GitHub 日报`

## 最终校验规则
- [ ] 标题清晰可读，居中或偏左
- [ ] 无人物
- [ ] 留白面积 > 40%
- [ ] 装饰元素不超过 2 种
- [ ] 整体感觉干净专业
