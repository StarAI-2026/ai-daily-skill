# 章节配图风格：扁平插画

> **风格定位**：简洁扁平、色彩明快、图形化表达。适合大众科普、轻松阅读。
> **核心原则**：图像主导，无人物，扁平化设计。

## 核心规范

| 维度 | 规范 |
|------|------|
| 画面比例 | 16:9 横版 |
| 无人物 | 画面中完全无人物 |
| 核心主体 | 扁平化图形呈现章节关键内容 |
| 文字 | 1-2 个关键词点睛 |
| 配色 | 与封面同色系，明快但不刺眼 |
| 阴影 | 无渐变阴影，纯色块为主 |

## 4 种内容图类型

### 类型 1：事件 / 过程
- 视觉：扁平化分步流程图，箭头连接，每步配简单图标
- 示例：工具链工作流、安全升级流程

### 类型 2：人物 / 产品
- 视觉：扁平化产品图解（设备/模型/架构的抽象图形）
- 示例：新模型发布、芯片架构

### 类型 3：数据 / 对比
- 视觉：扁平化柱状图/饼图/对比图，关键数字放大
- 示例：Benchmark 评测、市场份额

### 类型 4：场景 / 概念
- 视觉：扁平化场景插画（齿轮/火箭/灯泡等概念图形）
- 示例：宏观趋势、观点评论

## Prompt 模板

```
一张横版16:9扁平插画风格配图，{配色方案描述}。
核心主体：{根据内容类型生成的图形描述}。
背景：{纯色或简单纹理}。
文字：仅有1-2个英文关键词"{关键词}"做点睛。
无人物，无写实元素，全部为扁平化矢量风格图形。
```

## 重要约束
- 画面中不出现任何人物
- 不使用渐变/3D/写实效果
- 装饰元素不超过 2 种
- 配色与封面同色系
