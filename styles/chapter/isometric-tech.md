# 章节配图风格：等距科技

> **风格定位**：等距视角、3D几何、科技蓝调。适合技术深度、工程类内容。
> **核心原则**：图像主导，无人物，等距投影风格。

## 核心规范

| 维度 | 规范 |
|------|------|
| 画面比例 | 16:9 横版 |
| 无人物 | 画面中完全无人物 |
| 核心主体 | 等距视角的3D几何图形/模块/管线 |
| 文字 | 1-2 个关键词，小字标注 |
| 配色 | 深色底 + 发光线条，与封面同色系 |
| 视角 | 30度等距投影（isometric） |

## 4 种内容图类型

### 类型 1：事件 / 过程
- 视觉：等距视角的管线/流程，节点用3D方块表示，连线发光
- 示例：CI/CD流程、数据处理管线

### 类型 2：人物 / 产品
- 视觉：等距视角的产品架构图，3D模块堆叠
- 示例：模型架构、系统设计

### 类型 3：数据 / 对比
- 视觉：等距视角的3D柱状图/数据塔
- 示例：性能对比、规模展示

### 类型 4：场景 / 概念
- 视觉：等距视角的微缩场景（服务器机房/城市/网络）
- 示例：基础设施、生态全景

## Prompt 模板

```
一张横版16:9等距科技风格配图，30度等距投影视角，{配色方案描述}。
核心主体：{根据内容类型生成的3D几何图形描述}。
背景：{深色底，微弱网格}。
文字：仅有1-2个英文关键词"{关键词}"小字标注。
无人物，全部为等距视角的3D几何模块和发光线条。
```

## 重要约束
- 画面中不出现任何人物
- 必须是等距投影视角
- 发光效果克制，不喧宾夺主
- 配色与封面同色系
