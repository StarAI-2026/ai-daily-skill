# 章节配图风格：水彩柔和

> **风格定位**：手绘水彩、柔和色调、艺术感。适合人文向、评论类内容。
> **核心原则**：图像主导，无人物，水彩手绘风格。

## 核心规范

| 维度 | 规范 |
|------|------|
| 画面比例 | 16:9 横版 |
| 无人物 | 画面中完全无人物 |
| 核心主体 | 水彩手绘风格的图形/场景 |
| 文字 | 1-2 个关键词，手写体 |
| 配色 | 柔和色调，与封面同色系但更淡 |
| 笔触 | 水彩晕染、边缘柔和 |

## 4 种内容图类型

### 类型 1：事件 / 过程
- 视觉：水彩手绘的分步图，箭头为手绘曲线
- 示例：工具使用流程、操作步骤

### 类型 2：人物 / 产品
- 视觉：水彩手绘的产品/概念图，柔和的几何形状
- 示例：新工具发布、概念模型

### 类型 3：数据 / 对比
- 视觉：水彩手绘的图表，色块柔和
- 示例：市场趋势、增长数据

### 类型 4：场景 / 概念
- 视觉：水彩手绘的抽象场景（风景/城市/自然意象）
- 示例：趋势展望、行业观点

## Prompt 模板

```
一张横版16:9水彩柔和风格配图，{配色方案描述，柔和淡雅}。
核心主体：{根据内容类型生成的水彩手绘图形描述}。
背景：{水彩晕染底色，留白}。
文字：仅有1-2个手写体关键词"{关键词}"。
无人物，全部为水彩手绘风格，边缘柔和，有晕染效果。
```

## 重要约束
- 画面中不出现任何人物
- 必须是水彩手绘风格
- 色调柔和，不使用高饱和色
- 配色与封面同色系但降低饱和度
